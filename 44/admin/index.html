📱 Devices Beheer

🤖 AUTOMATISCH SYSTEEM: Alle devices werken automatisch!
✅ Devices via abonnement: Werken automatisch - worden automatisch geactiveerd bij actief abonnement
✅ Devices via admin panel: Werken automatisch - zijn altijd actief
❌ Devices kunnen NOOIT worden verwijderd (permanent systeem)
🔧 Devices via abonnement: Kunnen worden gecontroleerd (blokkeren/deblokkeren) via admin panel
🔒 Devices via admin panel: Zijn ALTIJD actief - kunnen niet worden geblokkeerd

💡 Eenvoudig blokkeren: Klik op "🚫 Blokkeren" om een device direct te blokkeren. Dit device krijgt dan geen internet toegang meer. Gebruik "✅ Deblokkeren" om toegang te herstellen. Je kunt ook de status dropdown gebruiken voor meer opties.

➕ Device Toevoegen (Super Eenvoudig - 1 Klik!)

🚀 Super Eenvoudig:
✅ Selecteer gebruiker
✅ Klik op "Device Toevoegen"
✅ Klaar! Alles wordt automatisch gegenereerd (naam, WireGuard key, IP)

🔒 Permanent Device: Dit device werkt altijd, ook zonder abonnement. Kan NOOIT worden verwijderd.

💡 Volledig Automatisch: Alleen gebruiker selecteren, de rest wordt automatisch gegenereerd (naam, WireGuard key, IP). Device naam is optioneel - wordt automatisch gegenereerd als leeg.

⚡ Automatisch Device Toevoegen

✅ Automatisch geselecteerd: De eerste gebruiker is automatisch geselecteerd!
Klik direct op "⚡ Automatisch Toevoegen" - alles wordt automatisch gegenereerd!

💡 Super snel: Device naam, WireGuard key en IP worden allemaal automatisch gegenereerd!

💡 Tip: Laat device naam leeg voor automatische detectie (iPhone, Android, Windows PC, etc.)
🔧 WireGuard key en IP worden automatisch gegenereerd - geen handmatige invoer nodig!
