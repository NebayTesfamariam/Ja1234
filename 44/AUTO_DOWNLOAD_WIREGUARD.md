# ✅ Automatische WireGuard Download

## 🎉 Wat is Nieuw

Na registratie wordt de WireGuard config **AUTOMATISCH** gedownload!

### ✅ Wat Gebeurt Er Nu:

1. **Registratie** → Account + Abonnement + Device aangemaakt
2. **Automatische Download** → WireGuard config wordt direct gedownload
3. **Instructies** → Stap-voor-stap instructies worden getoond
4. **Klaar!** → Importeer config in WireGuard app en activeer VPN

---

## 🔧 Hoe Het Werkt

### Token-Based Access
- Na registratie krijg je een `token`
- Dit token wordt gebruikt om WireGuard config te downloaden
- Geen extra login nodig!

### Automatische Download
- JavaScript triggert download automatisch
- Config wordt gedownload als `.conf` bestand
- Bestandsnaam: `wireguard-[device-name].conf`

---

## 📱 Instructies Na Download

### iPhone/iPad
1. Open **WireGuard** app
2. Klik **"+"** → **"Create from file or archive"**
3. Selecteer gedownloade `.conf` file
4. Activeer VPN verbinding

### Android
1. Open **WireGuard** app
2. Klik **"+"** → **"Create from file or archive"**
3. Selecteer gedownloade `.conf` file
4. Activeer VPN verbinding

### Windows/Mac/Linux
1. Open WireGuard client
2. Importeer `.conf` file
3. Activeer VPN verbinding
