## 🚀 Usage

### Enable Professional Features
The professional features are automatically enabled when:
1. `js/admin-pro.js` is loaded
2. `css/admin-pro.css` is loaded
3. Page loads

### Use Keyboard Shortcuts
- Press `Ctrl/Cmd + K` to focus search
- Press `Ctrl/Cmd + N` to create new item
- Press `1-9` to switch tabs
- Press `Esc` to close modals

### Use Bulk Operations
1. Select multiple items with checkboxes
2. Bulk actions bar appears automatically
3. Choose action (delete, make admin, etc.)
4. Confirm action

### Export Data
```javascript
// Export users as CSV
exportData('users', 'csv')

// Export devices as JSON
exportData('devices', 'json')
```
