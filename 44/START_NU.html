🚀 Start Nu - 5 Minuten Setup

Volg deze stappen om het systeem te gebruiken

📋 Stap 1: Database Setup

1

Database Aanmaken

Klik op de knop hieronder om de database automatisch aan te maken:

🔧 Setup Database

✅ Database wordt automatisch aangemaakt met alle tabellen

📋 Stap 2: DNS Server Starten

2

DNS Server Starten

Open terminal en voer uit:

cd /Applications/XAMPP/xamppfiles/htdocs/44
sudo python3 dns_whitelist_server.py

⚠️ Let op: Vereist sudo (root toegang) voor poort 53

✅ DNS server draait nu op poort 53

📋 Stap 3: Abonnement Aansluiten

3

Account Aanmaken

Klik op de knop hieronder om een abonnement aan te sluiten:

💰 Abonnement Aansluiten

Direct na aanmelding:

  • Abonnement is direct actief
  • Device wordt automatisch aangemaakt
  • Pornografische content wordt meteen geblokkeerd

📋 Stap 4: WireGuard Config Downloaden

4

VPN Configuratie

Log in en download je WireGuard config:

🔐 Inloggen

Na inloggen:

  1. Ga naar "Devices"
  2. Klik op "Download WireGuard Config"
  3. Importeer config in WireGuard app
  4. Activeer VPN verbinding

📱 Device Setup

iPhone/iPad

  1. Installeer WireGuard app (App Store)
  2. Open app → "+" → "Create from file"
  3. Selecteer gedownloade .conf file
  4. Activeer VPN

Android

  1. Installeer WireGuard app (Google Play)
  2. Open app → "+" → "Create from file"
  3. Selecteer gedownloade .conf file
  4. Activeer VPN

Windows/Mac/Linux

  1. Installeer WireGuard client
  2. Importeer .conf file
  3. Activeer VPN

✅ Verificatie

Test 1: Pornografische Content Geblokkeerd?

Probeer te bezoeken: pornhub.com (met VPN actief)

✅ Site laadt niet → ✅ Werkt!

Test 2: Whitelist Werkt?

Voeg toe: wikipedia.org aan whitelist

Probeer te bezoeken

✅ Site laadt wel → ✅ Werkt!

🆘 Problemen?

DNS Server Start Niet?

sudo python3 dns_whitelist_server.py

VPN Verbindt Niet?

  • Check WireGuard config: AllowedIPs = 0.0.0.0/0
  • Check DNS: DNS = 10.10.0.1

Geen Internet?

Voeg domeinen toe aan whitelist (google.com, wikipedia.org, etc.)
